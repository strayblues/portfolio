A multiple choice quiz app for mobile devices.

The quiz contains 10 randomly selected questions. There is no time limit. You get 1 point for each correct answer. At the end of the test, your total score is calculated and displayed.

Installation

In order to install the .apk file on your Android device, go to:

Settings > General > Security > Unknown sources

and allow installation from unknown sources.
